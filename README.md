# Ansible Localhost Ping Test

This example demonstrates a basic Ansible ad-hoc command to verify a successful Ansible installation. It pings localhost and gathers system facts, illustrating Ansible's agentless approach to managing local or remote machines. This is a common first step after installing Ansible, confirming the tool's readiness for automation tasks.

## Language

`bash`

## How to Run

1. Ensure Ansible is installed on your system (as per the accompanying article).
2. Save the code as `ansible_test.sh`.
3. Make the script executable: `chmod +x ansible_test.sh`.
4. Run it from your terminal: `./ansible_test.sh`
